# JT::Tools

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jt_tools', groups: [:development]
```

And then execute:

    $ bundle
    $ bin/rails jt_tools:install

## Usage

```bash
bin/pronto run
```

```bash
bin/rubocop
```

```bash
bin/tools-setup
```

```bash
bin/tools-upgrade
```

```bash
bin/lint-github-pr
```
